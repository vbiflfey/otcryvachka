# Открывашка

Откроет тебе всё, что постоянно закрывается. 

## Возможности

-  Автоматическое отслеживание процессов (проверка каждые 5 секунд)
-  Автоматический перезапуск закрытых приложений
-  Поддержка .exe, .bat, .cmd, .ps1 файлов
-  Простой интерфейс с выбором файлов через проводник
-  Автозапуск вместе с Windows
-  Сохранение списка приложений в JSON

## Установка и запуск

### Способ 1: Из исходников (требуется Python)

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск
python otkryvashka.py

### Способ 2: Готовый .exe файл

Скачайте последний релиз

Распакуйте архив

Запустите Otkryvashka.exe

Как использовать
Нажмите "Добавить приложение" и выберите нужные .exe, .bat, .cmd или .ps1 файлы

Программа автоматически начнет следить за ними

Если приложение закроется - оно будет автоматически перезапущено

Нажмите "Включить автозапуск", чтобы Открывашка запускалась с Windows
